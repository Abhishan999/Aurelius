# 💼 Aurelius Finance Dashboard

Aurelius Finance is a modern, responsive financial dashboard built using **React (via CDN)** and **Chart.js**, designed to help users track income, expenses, and financial insights in a clean and interactive UI.

---

## 🚀 Features

### 📊 Dashboard Overview
- Total Balance, Income, Expenses, and Savings Rate
- Monthly Income vs Expense trends (Line Chart)
- Category-wise spending breakdown (Donut Chart)

### 💳 Transactions Management
- Add, Edit, Delete transactions (Admin mode)
- Search and filter by:
  - Type (Income / Expense)
  - Category
- Sorting by date and amount
- Pagination support

### 📈 Insights Page
- Smart financial insights:
  - Top spending category
  - Savings rate analysis
  - Monthly expense comparison
- Visual breakdowns and progress indicators

### 🎨 UI/UX
- Dark & Light mode toggle
- Responsive design (mobile-friendly)
- Elegant glassmorphism-inspired UI
- Smooth animations and transitions

### ⚙️ State Management
- Built using **React Context API + useReducer**
- Persistent data using **LocalStorage**

### 📤 Export Feature
- Export all transactions as a CSV file

---

## 🛠️ Tech Stack

- **Frontend:** React (CDN + Babel)
- **Charts:** Chart.js
- **Styling:** Pure CSS (custom design system)
- **State Management:** React Context API
- **Storage:** LocalStorage
